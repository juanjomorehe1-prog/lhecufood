# 🥗 Constructor de Ensaladas

Aplicación web interactiva para crear ensaladas personalizadas y enviar el pedido por WhatsApp.

## ✨ Características

- Selección intuitiva de ingredientes por categorías
- Cálculo automático de calorías
- Interfaz moderna con diseño glassmorphism
- Integración directa con WhatsApp
- Totalmente responsive

## 🚀 Demo

Visita la aplicación en: `https://[tu-usuario].github.io/constructor-ensaladas/`

## 📱 Uso

1. Selecciona tus ingredientes favoritos
2. Revisa el contador de calorías
3. Haz clic en "Enviar a WhatsApp"
4. ¡Listo! Se abrirá WhatsApp con tu pedido

## 🛠️ Tecnologías

- HTML5
- CSS3 (Glassmorphism, Animaciones)
- JavaScript Vanilla
